# TimeCog - 시간 인지 능력 테스트

ADHD 학생의 시간 경과 추정 능력 향상을 위한 인터랙티브 미니게임입니다.

## 핵심 원리

### ADHD 학생의 특성
- 시간 경과 추정 능력 약함 → 조직화 기술 기반 개입

### PET 전략 반영
- **P** (Pause): 잠깐 멈추기
- **E** (Evaluate): 평가하기
- **T** (Trust): 신뢰하기

## 주요 기능

### ⏱️ 시간 감각 테스트
- 10초 타이머 게임
- 학생이 직접 정지 버튼으로 시간 측정
- 실제 시간과의 차이 자동 표시

### 📊 시각화된 결과
- 3회 반복 후 평균 오차 계산
- "내가 느낀 시간 vs 실제 시간" 비교 차트
- 경향성 분석 (빠름/느림/정확)

### 🤔 자기성찰
- "내가 시간이 짧게/길게 느껴지는 편인가?"
- "과제할 때 이 감각이 어떤 영향을 줄까?"

## 디자인 스타일

### 무인양품(MUJI) 스타일
- 미니멀하고 깔끔한 디자인
- ADHD 학생의 집중력을 방해하지 않는 UI
- 차분한 색상 팔레트

### 색상 시스템
- `muji-dark`: #3a3a3a (주요 텍스트, 버튼)
- `muji-mid`: #6b6b6b (보조 텍스트)
- `muji-beige`: #e8e3db (테두리, 배경 강조)
- `muji-bg`: #f5f3ef (메인 배경)

## 기술 스택

- React 18 - UI 라이브러리
- TypeScript - 타입 안정성
- Vite - 빌드 도구
- Tailwind CSS - 스타일링
- Zustand - 상태 관리 (localStorage 자동 저장)
- Framer Motion - 애니메이션
- Recharts - 차트 시각화

## 설치 및 실행

### 필수 요구사항
- Node.js 18 이상
- npm 또는 yarn

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```
브라우저에서 http://localhost:5173 을 열어 확인할 수 있습니다.

### 프로덕션 빌드
```bash
npm run build
```
빌드된 파일은 `dist` 폴더에 생성됩니다.

### 프리뷰
```bash
npm run preview
```

## 게임 플로우

1. **인트로 화면**: PET 전략 소개, 게임 방법 안내
2. **라운드 준비**: 각 라운드 시작 전 준비 화면
3. **게임 진행**: 10초 타이머, 학생이 정지 버튼 클릭
4. **라운드 결과**: 즉각적인 피드백과 오차 표시
5. **최종 결과**: 3회 결과 종합, 차트로 시각화
6. **자기성찰**: 시간 감각과 과제 영향 질문

## 교육적 효과

✅ 자기점검을 통한 메타인지 향상
✅ 시각적 피드백으로 즉각적 학습
✅ 반복 연습으로 시간 감각 개선
✅ 자기성찰을 통한 인식 전환

## 데이터 저장

- localStorage를 통해 자동 저장
- 페이지 새로고침 후에도 데이터 유지
- 개인정보 수집 없음

## 향후 확장 계획

- 다양한 목표 시간 (5초, 30초, 1분 등)
- 난이도 조절 기능
- 연습 기록 및 추이 그래프
